# Feature Delivery Workflow

Ne zaman kullanılır:
- Orta/küçük ölçekli yeni özellik geliştirmeleri

Adımlar:
1. Özelliği 1-3 küçük dilime böl.
2. Her dilim için test yaz.
3. Dilimi kodla, testleri geçir.
4. Sonunda bütünleştirici testleri çalıştır.

Teslim kontrolü:
- Değişen dosyalar listesi
- Davranış farkı özeti
- Çalıştırılan test komutları
- Bilinen eksik/kısıtlar
