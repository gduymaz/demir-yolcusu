# Mobile Performance (Godot)

Ne zaman kullanılır:
- FPS düşüşü, bellek artışı, yükleme gecikmesi

Adımlar:
1. Ölçüm hedefi koy (FPS, memory, load time).
2. Darboğazı bul (scene, script, texture, draw call).
3. Minimum riskli optimizasyon uygula.
4. Önce/sonra ölçümünü raporla.

Çıktı:
- Ölçüm metrikleri
- Uygulanan optimizasyonlar
- Davranış regresyon durumu
